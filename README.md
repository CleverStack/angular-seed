## Install

Install is made easy using the CleverStack CLI, first install [node.js](http://nodejs.org)

1. `$ npm install -g cleverstack-cli`
2. `$ clever init my-app` && `$ cd my-app`
4. `$ clever serve`
5. Go to: `http://localhost:9000`
